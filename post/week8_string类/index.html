Week8_string类

Apr 7, 2020
1 min read

一、string 类是模板类：

typedef basic_string<char> string;

//使用string类要包含头文件 <string>
//string对象的初始化：
string s1("Hello"); 
string month = "March"; 
string s2(8,’x’);       //s2中有8个x

//错误的初始化方法：
string error1 = ‘c’; // 错 
string error2(‘u’); // 错 
string error3 = 22; // 错 
string error4(8); // 错 

//可以将字符赋值给string对象
string s;
s = ‘n’;

string类程序样例

#include <iostream>
#include <string>
using namespace std;
int main(int argc, char* argv[ ]){
	string s1("Hello");
	cout << s1 << endl;
	string s2(8,'x');
	cout << s2 << endl;
	string month = "March";
	cout << month << endl;
	string s;
	s='n';
	cout << s << endl;
	return 0;
}
