一、基类和派生类指针的互相转换

（1）公有派生的情况下,派生类对象的指针可以直接赋值给基类指针

（2）基类的指针不能赋值给派生类的指针，除非强制类型转换，

Base * ptrBase = &objDerived;//本来就指向一个派生类的对象
Derived *ptrDerived = (Derived * ) ptrBase;

注意 ：需要保证被转换的基类指针本来就指向一个派生类的对象。

注意2 ：即便基类指针指向的是一个派生类对象，也不能通过基类指针访问基类没有而派生类中有的成员。
